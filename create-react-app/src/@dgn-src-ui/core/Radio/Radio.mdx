# Radio

```js static
import Radio from "@dgn-src-ui/core/Radio/Radio";
```

## Usage

```jsx
<Radio size="large" />
```

`Radio` can also be a control value within a `FormControlLabel` Component. The text in the `label` prop adds text next to the radio button that when clicked, also sets the radio select state.

```jsx
<FormControlLabel control={<Radio size="large" />} label="Option 1" />
```

`Radio` is often in a `RadioGroup` Wrapper Component that allows to set a single `onChange` prop, includes proper keyboard accessibility to the group of Radios.

```jsx
<RadioGroup value={value} onChange={handleChange}>
  <FormControlLabel
    label={label}
    control={<Radio size={size} />}
    labelPlacement="end"
    value="a"
  />
  <FormControlLabel
    label={label}
    control={<Radio size={size} />}
    labelPlacement="end"
    value="b"
  />
</RadioGroup>
```
