# Link

```js static
import Link from "@dgn-src-ui/core/Link/Link";
import FullWidthLink from "@dgn-src-ui/core/Link/FullWidthLink/FullWidthLink";
```

## Usage

The link colors will always be the secondary color of the chosen theme.

```jsx
<Link href="#">Click Here</Link>
<FullWidthLink href="#">Click Here</FullWidthLink>
```

## Props

<Props of={Link} />

## Notes

- `Link` is used for inline links
- `FullWidthLink` is used for any centered links
