# Form

```js static
import Form from "@dgn-src-ui/core/Form";
```

## Usage

`Form` is intended to be a wrapper for all form components (eg. textfields, submit form buttons)

```jsx
<Form>
  <TextField type="tel" id="cardNumber" label="Card Number" />
</Form>
```
