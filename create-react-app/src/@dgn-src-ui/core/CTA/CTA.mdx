# CTA Component

```js static
import CTA from "@dgn-src-ui/core/CTA/CTA";
```

## Usage

`CTA` is intended to be a wrapper for user navigation components

```jsx
<CTA>
  <Button>Continue</Button>
  <FullWidthLink>Cancel and return to merchant</FullWidthLink>
</CTA>
```
