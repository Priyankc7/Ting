# Carousel

```js static
import Carousel from "@dgn-src-ui/core/Carousel/Carousel";
```
## Usage

```Carousel``` component is built with the ```MobileStepper``` component, which uses the ```Paper``` and the ```Stepper``` APIs to move between views. The React Swipeable Views library is used to create the sliding Carousel effect.

```jsx
<Carousel slides={slides} onCarouselUpdate={slideAction} />
```
