# Select

```js static
import Select from "@dgn-src-ui/core/Select";
```

## Usage

```jsx
<Select id="fieldName" name="fieldName">
  <Option value="">
    <em>None</em>
  </Option>
  <Option value={10}>Ten</Option>
  <Option value={20}>Twenty</Option>
  <Option value={30}>Thirty</Option>
</Select>
```
