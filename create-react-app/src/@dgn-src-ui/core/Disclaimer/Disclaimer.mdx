# Disclaimer Component

```js static
import Disclaimer from "@dgn-src-ui/core/Disclaimer/Disclaimer";
```

## Usage

`Disclaimer` is intended to be a wrapper for disclaimer statements and links

```jsx
<Disclaimer>
  <Typography variant="body2">
    Your information will be shared with participating payment networks,
    service providers and as otherwise described in our Privacy Notice to give
    you the appropriate experience.
  </Typography>
</Disclaimer>
```
