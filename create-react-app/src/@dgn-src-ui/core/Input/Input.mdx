# Input

```js static
import Input from "@dgn-src-ui/core/Input/Input";
```

## Usage

- `Input` is a wrapper for material ui input components (select, text field, etc.)
- Adds an input label, and optional validation icons

```jsx
<Input>
  <FilledInput fullWidth />
</Input>
```
