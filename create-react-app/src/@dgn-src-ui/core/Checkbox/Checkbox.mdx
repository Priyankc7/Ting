# Checkbox

```js static
import Checkbox from "@dgn-src-ui/core/Checkbox/Checkbox";
```

## Usage

A ```Checkbox``` is used for multi-select, or when a user should have the opportunity to deselect an option. This is opposed to a ```Radio``` button, which does not allow for deselecting or multi-select.

```jsx
<Checkbox name="rememberDevice" />
```

```Checkbox``` can also be a control value within a ```FormControlLabel``` Component. The text in the ```label``` prop adds text next to the checkbox that when clicked, also toggles checkbox select state.
```jsx
<FormControlLabel
  control={<Checkbox name="rememberDevice" />}
  label="Remember me on this device"
/>
```
