# Card Components

```js static
import Card from "@dgn-src-ui/core/Card/Card";
import CardArt from "@dgn-src-ui/core/CardArt/CardArt";
```

## Usage

### Card

`Card` is a wrapper for `CardArt`

```jsx
<Card title="Pay With">
  <CardArt src={"/assets/images/it-card-pride-front@3x.jpg"} />
  <Typography variant="body2">
    Discover Card <br />
    Ending in 9101 <br />
    montgomerySmith@gmail.com
  </Typography>
</Card>
```

### CardArt

```jsx
<CardArt src={src} alt={alt} />
```
