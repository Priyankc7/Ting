# Notification Page Components

```js static
import NotificationPage from "@dgn-src-ui/core/NotificationPage/NotificationPage";
import NotificationPageIcon from "@dgn-src-ui/core/NotificationPageIcon/NotificationPageIcon";
import NotificationPageHeading from "@dgn-src-ui/core/NotificationPageHeading/NotificationPageHeading";
import NotificationPageContent from "@dgn-src-ui/core/NotificationPageContent/NotificationPageContent";
```

## Usage

### NotificationPage

`NotificationPage` is a wrapper for `NotificationPageIcon`, `NotificationPageHeading`, and `NotificationPageContent`

```jsx
<NotificationPageComp>
  <NotificationPageIcon />
  <NotificationPageHeading />
  <NotificationPageContent />
</NotificationPageComp>
```

### NotificationPageIcon

```jsx
<NotificationPageIcon Icon={CardUnlockedIcon} />
```

### NotificationPageHeading

`NotificationPageHeading` is a wrapper for the page heading

```jsx
<NotificationPageHeading>Unlock your account</NotificationPageHeading>
```

### NotificationPageContent

```jsx
<NotificationPageContent>
  <Typography>
    Send a one-time code to your email to access your account.
  </Typography>
  <CTA>
    <Button>Send Code</Button>
    <FullWidthLink>Cancel and return to merchant</FullWidthLink>
  </CTA>
</NotificationPageContent>
```
