## Form Fields

### Address Fields

| Field Name | Rules    | Description                                                              |
| ---------- | -------- | ------------------------------------------------------------------------ |
| name       | required | Full name of consumer, only required if no firstName and lastName fields |
| firstName  | required | First name of consumer, only required if no name fields                  |
| lastName   | required | Last name of consumer, only required if no name fields                   |
| line1      | required | Address line 1 of shipping address                                       |
| line2      | required | Address line 2 of shipping address                                       |
| line3      | required | Address line 3 of shipping address                                       |
| city       | required | City of shipping address                                                 |
| state      | required | State or province of shipping address                                    |
| zip        | required | Zip Code or Postal Code of shipping address                              |
