# Review and Confirm Screen

On this screen, the consumer has the opportunity to review their information before submitting. They will be required to select a shipping address before being allowed to continue. From this screen they have multiple actions, provided by callback methods.

## Usage

```jsx
import View from "@dgn-src-ui/core/View";

// ...

<View
  screen="ReviewAndConfirm"
  addresses={[
    {
      addressId: 12,
      name: "Phoebe Buffay",
      line1: "5 Morton St",
      city: "New York",
      state: "NY",
      zip: "10014",
    },
  ]}
  card={{
    type: "Discover Card",
    number: "9101",
    contact: "mo*******ith@gmail.com",
    image: "https://cardart.assets.discover.com/it-card-pride-front@3x.jpg",
  }}
  form={{
    onSubmit: (data, e, context) => {
      // Handle submission
    },
  }}
  onProfileMenuItemClick={(val, e, data) => {
    // Handle profile menu item click
  }}
  onManageCardsMenuItemClick={(val, e, data) => {
    // Handle manage cards menu item click
  }}
  onShippingMenuItemClick={(val, e, data) => {
    // Handle shipping menu item click
  }}
  onCancelClick={(e) => {
    e.preventDefault();
    // Navigate back to merchant
  }}
/>;
```

<PassingViewProps />

## Form Fields

| Field Name     | Rules    | Description                                                  | Default |
| -------------- | -------- | ------------------------------------------------------------ | ------- |
| addressId      | required | Contains the ID of the shipping address they want to ship to |         |
| rememberDevice |          | Whether or not this device should be remembered              | true    |
