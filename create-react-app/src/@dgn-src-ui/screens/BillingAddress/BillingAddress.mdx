# Enter Billing Address Screen

This screen handles the edge case for when the API does not return a billing address for the consumer. They must be prompted with this form to enter a billing address before being sent to the Review and Confirm screen.

## Usage

```jsx
import View from "@dgn-src-ui/core/View";

// ...
<View
  screen="BillingAddress"
  form={{
    onSubmit: (data, e, context) => {
      // Handle submission
    },
    defaultValues: {
      firstName: "Montgomery",
      lastName: "Smith",
      line1: "123, NW 45th ST",
      state: "NY",
      city: "New York",
      zip: "12345",
      countryCode: "US",
    },
  }}
  onCancelClick={(e) => {
    e.preventDefault();
    // Navigate back to merchant
  }}
/>;
```
