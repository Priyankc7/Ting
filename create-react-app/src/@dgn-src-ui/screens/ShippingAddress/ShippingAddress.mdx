# Shipping Address Screen

This screen handles both the add and edit shipping address screens. Providing default values to the `form` prop object will automatically trigger loading of the "Edit Shipping Address" version of the screen.

## Usage

```jsx
import View from "@dgn-src-ui/core/View";

// ...

<View
  screen="ShippingAddress"
  form={{
    onSubmit: (data, e, context) => {
      // Handle submission
    },
    defaultValues: {
      addressId: 5,
      name: "Montgomery Smith",
      line1: "123 NW 45th ST",
      city: "New York",
      state: "NY",
      zip: "10001",
      countryCode: "US",
    },
  }}
  onCancelClick={(e) => {
    e.preventDefault();
    // Navigate back to merchant
  }}
/>;
```
