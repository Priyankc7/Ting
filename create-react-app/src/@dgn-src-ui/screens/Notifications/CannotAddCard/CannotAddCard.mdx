# Cannot Add Card Screen

For a number of reasons, the consumer may not be able to add their card to the screen. If they cannot, there are a number of variations of this screen that can be shown.

## Usage

```jsx
import View from "@dgn-src-ui/core/View";

// ...

<View screen="Notifications/CannotAddCard" />;

// or for a specific "CannotAddCard" screen
<View screen="Notifications/CannotAddCard/DuplicateCard" />;
```
