# Account Locked Screen

This screen is used after the consumer has attempted to access their account too many times and have been locked out of their account.

## Usage

```jsx
import View from "@dgn-src-ui/core/View";

// ...

<View
  screen="Notifications/AccountLocked"
  timeout={300}
  onTimeout={() => {
    // Do something on timeout, like navigate
    // to screen 4.2.0 Unlock Account
  }}
/>;
```
