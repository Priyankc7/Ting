# Remove Card Screen

This screen is the confirmation screen for removing the current card from the consumer's wallet, from the Review and Confirm page.

## Usage

```jsx
import View from "@dgn-src-ui/core/View";

// ...

<View
  screen="Confirmations/RemoveCard"
  card={{
    image: "https://assets.discovernetwork.com/it-card.png",
    type: "Discover Card",
    lastFour: "9201",
  }}
  form={{
    onSubmit: (data, e, context) => {
      // Handle submission
    },
  }}
  onCancelClick={(e) => {
    e.preventDefault();
    // Navigate back to review and confirm page
  }}
/>;
```
