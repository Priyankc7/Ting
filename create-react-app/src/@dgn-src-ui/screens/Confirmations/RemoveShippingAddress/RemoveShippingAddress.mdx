# Delete Shipping Address Screen

This screen is the confirmation screen for deleting a shipping address from the consumer.

## Usage

```jsx
import View from "@dgn-src-ui/core/View";

// ...

<View
  screen="Confirmations/RemoveShippingAddress"
  address={{
    addressId: "a33a-asd3-hSf4",
    name: "Montgomery Smith",
    line1: "123 NW 45th ST",
    city: "New York",
    state: "NY",
    zip: "12345",
  }}
  form={{
    onSubmit: (data, e, context) => {
      // Handle submission
    },
  }}
  onCancelClick={(e) => {
    e.preventDefault();
    // Navigate back to shipping address
  }}
/>;
```
